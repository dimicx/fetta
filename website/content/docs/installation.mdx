# Installation

Install fetta using your preferred package manager:

<Tabs items={['pnpm', 'bun', 'npm', 'yarn']}>
  <Tab value="pnpm">
```bash
pnpm add fetta
```
  </Tab>
  <Tab value="bun">
```bash
bun add fetta
```
  </Tab>
  <Tab value="npm">
```bash
npm install fetta
```
  </Tab>
  <Tab value="yarn">
```bash
yarn add fetta
```
  </Tab>
</Tabs>

## Requirements

- **Node.js** 18+
- **React** 18+ (for React component, optional)

## Browser Support

fetta requires these modern browser features:

- `ResizeObserver`
- `Intl.Segmenter`
- `Range.getBoundingClientRect()`

All evergreen browsers are supported (Chrome, Firefox, Safari, Edge).
